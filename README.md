# Augweb

Web UI for [auger](http://github.com/brewster/auger).

## What's it look like?

![Augweb Screenshot](https://raw.github.com/brewster/assets/master/img/auger/web_screenshot.png "Augweb")

## How do I run it?

    bundle install
    ln -s your/auger/cfg cfg
    rackup
    open http://localhost:9292
